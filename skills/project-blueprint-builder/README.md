# Project Blueprint Builder Skill

This package contains a reusable skill for planning any software project from idea to implementation. It is designed to be practical, structured, and easy to share as a publishable skill package.

## What is included

- A main skill definition in SKILL.md
- Planning templates for:
  - PRD.md
  - ARCHITECTURE.md
  - RULES.md
  - PHASES.md
  - DESIGN.md
  - MEMORY.md

## How to use it

Open the skill in an agent or skills catalog and ask for a project blueprint. Example prompts include:

- Create a full blueprint for a SaaS analytics platform.
- Generate PRD, architecture, rules, phases, design, and memory docs for a mobile app.
- Plan an AI-powered internal productivity tool from idea to implementation.

## GitHub publishing

To publish this package on GitHub:

1. Create a repository on GitHub.
2. Upload this folder as the repository content.
3. Make sure the repository root contains SKILL.md.
4. Share the repository URL or archive link with your preferred skill marketplace.
