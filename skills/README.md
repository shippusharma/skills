# Skills package

This folder contains a reusable skill for planning software projects from scratch.

## Included skill

- project-blueprint-builder

## Files

- SKILL.md
- PRD.md
- ARCHITECTURE.md
- RULES.md
- PHASES.md
- DESIGN.md
- MEMORY.md

## How to use it

1. Open the skill folder in your agent or skills catalog.
2. Run the skill with a request such as: "Create a complete project blueprint for a SaaS dashboard."
3. The skill will generate the planning documents in the project directory.

## How to share on skills.sh

1. Publish this folder to a Git repository or archive.
2. Import or upload it to skills.sh using the platform flow.
3. Make sure the root contains SKILL.md so the package is recognized.
